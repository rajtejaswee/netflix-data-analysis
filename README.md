# 📊 Netflix Data Analysis (EDA Project)

This project focuses on performing Exploratory Data Analysis (EDA) on a dataset containing Netflix’s content catalog. The goal is to analyze patterns in content types, release trends, ratings, countries, and more — using Python, Pandas, Matplotlib, and Seaborn.

Through this project, I aim to:
- Strengthen my data wrangling and visualization skills
- Extract actionable insights from real-world datasets
- Build a clean, industry-standard portfolio project as a Data Analyst

🛠️ Tools & Libraries Used:
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebooks

📁 Dataset:
Netflix Movies and TV Shows Dataset – available on Kaggle.

📌 Key Tasks:
- Data Cleaning and Preprocessing
- Handling missing values and duplicates
- Univariate and Bivariate Analysis
- Visual storytelling with clean, insightful plots
- Documenting insights for portfolio/industry presentation
